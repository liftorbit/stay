# Conexões de módulos no ESP32

Este arquivo indica em quais pinos cada módulo é conectado, incluido os pinos de alimentação.

# Ignição motor principal

- VCC: G26

## LED status

- VCC: G25

## Módulo microSD

- VCC: 5v
- MISO: G19
- MOSI: G23
- SCK: G18
- CS: G5

## BMP280

- VCC: 3.3v
- SCL: G13
- SDA: G14
- CSB: G27
- SDO: G12

## BMI160

- VCC: 3.3v
- SCL: G22
- SDA: G21

## Sensor infravermelho

- VCC: 5v
- D0: G35
